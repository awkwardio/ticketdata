Hello world and welcome to my ticket data site!

My name is Chris and I'm a web developer & film buff. Since the summer of 2009, I've been collecting movie ticket stubs as a hobby. My total count so far is 132 ticket stubs! To further my education in programming and web development, I decided to put some data from this ticket collection into a neat little website with pie charts and graphs.

Here's what I did:

  1. I entered information about my tickets into a table in a MySQL Database
  2. I used a MySQL command to turn that data into an Excel spreadsheet
  3. I made pie charts and graphs using data from the Excel spreadsheet
  4. Profit! (just kidding, I'm not making money from this)

These pie charts and graphs were made using a Python library called matplotlib. If you want to take a look at how I did this, this website's code is available on my github page. I might also revisit this page from time to time and modify it to add more movies that I plan on seeing in the future. Anyway, thanks for stopping by!
